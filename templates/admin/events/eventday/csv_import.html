Import Event Schedule (CSV)

Upload a CSV file to bulk-import events. Existing events with the same date and event_type will be skipped (not overwritten).

Required columns

  • date — format: YYYY-MM-DD
  • event_type — one of: game, private_party, live_music, holiday, other

Optional columns

  • label — display label, e.g. Pirates vs. Cubs
  • team — one of: pirates, steelers, penguins, other
  • game_time — format: HH:MM (24-hour)
  • limited_menu — true or false (default: true)
  • limited_menu_lead_hours — integer, default 2
