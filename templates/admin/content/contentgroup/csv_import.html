Upload a CSV file to create content groups and their slots. Existing records are matched by slug and will not be overwritten. Safe to re-run with the same file.

CSV Format

Required columns: type, slug, label
Optional columns: description, component_type, order

Rows with type="group" define a content group. Rows with type="slot" define a slot belonging to the most recent group row above them.

type slug label description component_type order
group hero_banner Hero Banner Homepage hero panel
slot hero_banner_title Hero Title title 1
slot hero_banner_body Hero Body body 2

Valid component_type values: title, subtitle, body, footer, image, button
